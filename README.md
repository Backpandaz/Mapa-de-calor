**MAPA-DE-CALOR**

`Projeto criado para produzir um mapa de calor. É utilizado duas funções circulares, uma incrementada com um valor inteiro maior que 0 e a outra não. Fazendo a diferença entre as duas tem-se um degrade, uma vez que uma função está mais "densa" do que a outra. `

Ferramental:
Python, 
Matplotlib,
Numpy
